About Learning Nodes

Learning nodes are the individual steps in your roadmap. Each node should represent a specific concept, skill, or milestone that learners need to master.

Tips for creating effective nodes:
  • Focus on one concept per node
  • Provide clear explanations
  • Include practical examples when possible
  • Consider prerequisites for each step

After creating your node, you'll be able to add resource links to help learners master the content.

Formatting Tips

You can use markdown formatting in your description:

  • # Heading 1 for main headings
  • ## Heading 2 for subheadings
  • **bold text** for bold text
  • *italic text* for italic text
  • - item for bullet lists
  • 1. item for numbered lists
  • [link text](url) for links
