About Resource Links

Resource links provide additional learning materials for each node in your roadmap. These can include articles, videos, courses, books, tools, and more.

Tips for selecting resources:
  • Include a variety of resource types
  • Prioritize high-quality, up-to-date content
  • Consider both free and paid options
  • Include resources for different learning styles

Resource Types

  • Article Blog posts, tutorials, documentation
  • Video YouTube videos, conference talks
  • Course Online courses, workshops
  • Book Books, ebooks, guides
  • Tool Software tools, applications
  • Open Source GitHub repositories, open source projects
  • Other Any other type of resource
